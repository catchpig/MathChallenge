# 思维星球（Thinking Planet）设计规格文档

**日期**：2026-06-20
**平台**：Android 平板（横屏）
**目标用户**：小学女生（1-6年级）

---

## 1. 产品概述

### 1.1 目标

为小学一至六年级（含上下学期，共12个学期单元）的女生提供一款多元思维能力训练应用。通过沉浸式主题冒险世界 + 荣誉收集机制，让孩子在游戏化体验中提升数学、逻辑、空间、规律、创意五大思维维度。

### 1.2 核心特征

- 完全免费，无广告，无内购
- Android 平板横屏原生应用
- 单设备单用户，无需账号登录
- 内置题库 + 可扩展题包（JSON DLC）
- 荣誉收集驱动，无付费墙

---

## 2. 世界结构

### 2.1 六大主题世界（对应6个年级）

| 年级 | 世界名称 | 主题 emoji | 主色调 |
|------|---------|-----------|--------|
| 一年级 | 樱花仙境 | 🌸 | 粉红 #FFB7D5 |
| 二年级 | 独角兽草原 | 🦄 | 薰衣草紫 #D8B4FE |
| 三年级 | 人鱼海湾 | 🧜‍♀️ | 薄荷绿 #6EE7B7 |
| 四年级 | 甜心糖果城 | 🍭 | 蜜糖黄 #FDE68A |
| 五年级 | 云朵城堡 | ☁️🏰 | 天空蓝 #BAE6FD |
| 六年级 | 彩虹星球 | 🌈⭐ | 彩虹渐变 |

### 2.2 章节划分

每个世界分为两章：
- **第一章**：对应该年级上学期
- **第二章**：对应该年级下学期

共 **12 个学期单元**。

### 2.3 思维维度（每章内）

每个章节包含 5 个思维维度区域，分布在场景地图上：

| 维度 | 图标 | 内容 |
|------|------|------|
| 数学 | 🔢 | 计算、数感、运算规则 |
| 逻辑 | 🧩 | 推理、判断、因果关系 |
| 空间 | 🎯 | 图形变换、方向、几何 |
| 规律 | 🔍 | 数列、图案规律、找规律 |
| 创意 | 🎨 | 开放性思维、多解题目 |

### 2.4 解锁规则

- 完成当前章节 ≥ 60% → 解锁同世界下一章
- 完成同一世界两章各 ≥ 60% → 解锁下一个世界
- 已解锁范围内可自由选择维度和关卡
- 可重做任意关卡以提升星级

## 3. 题目设计

### 3.1 题目类型（5种）

| 类型 | 适用维度 | 交互方式 |
|------|---------|---------|
| 四选一选择题 | 全部 | 点击选项卡片 |
| 数字填空题 | 数学、规律 | 数字键盘输入 |
| 连线匹配题 | 逻辑、规律 | 拖拽连线 |
| 排序拖拽题 | 数学、逻辑 | 拖拽排序 |
| 图形选择题 | 空间、创意 | 点击图形选项 |

### 3.2 每关配置

- 每个维度内若干关卡，每关 5-10 道题
- 同一章节内难度由易到难递进
- 答错提供温柔提示（不批评），可选"再试一次"或"看答案"

### 3.3 题库格式（JSON）

```json
{
  "pack_id": "grade1_upper",
  "world": "sakura",
  "grade": 1,
  "semester": "upper",
  "dimensions": {
    "math": [
      {
        "id": "q001",
        "type": "choice",
        "question": "小美有24朵樱花，送给朋友8朵，还剩几朵？",
        "image": null,
        "options": ["14", "16", "18", "32"],
        "answer": "B",
        "hint": "24 - 8 = ?"
      }
    ],
    "logic": [],
    "spatial": [],
    "pattern": [],
    "creative": []
  }
}
```

内置题库存放于 `assets/packs/`，扩展题包以相同格式新增 JSON 文件。

## 4. 荣誉系统（四层）

### 第一层：关卡星级（即时反馈）

| 答对率 | 获得星级 |
|--------|---------|
| ≥ 60% | ⭐ 1星 |
| ≥ 80% | ⭐⭐ 2星 |
| 100% | ⭐⭐⭐ 3星 |

- 完成关卡立即展示星星动画
- 可重做关卡刷新星级（取最高记录）

### 第二层：维度勋章

每个思维维度各一枚勋章，铜/银/金三级：

| 平均星级 | 勋章等级 |
|---------|---------|
| 关卡平均 1.x 星 | 🥉 铜牌 |
| 关卡平均 2.x 星 | 🥈 银牌 |
| 关卡全部 3 星 | 🥇 金牌 |

每章共 5 枚维度勋章，全12章共 60 枚。

### 第三层：学期奖杯

每完成一个学期单元可获得奖杯，共 12 座：

| 条件 | 奖杯等级 |
|------|---------|
| 5个维度全部至少完成 | 🥉 铜奖杯 |
| 5个维度全部达到银牌+ | 🥈 银奖杯 |
| 5个维度全部达到金牌 | 🥇 金奖杯 |

获得金奖杯时触发全屏庆典动画（花瓣撒落 + 向导精灵跳舞）。

### 第四层：专属称号（动态授予）

| 称号 | 解锁条件 |
|------|---------|
| 🌸 数学小公主 | 数学维度金牌数最多 |
| 🦄 逻辑魔法师 | 逻辑维度金牌数最多 |
| 🧜‍♀️ 空间探险家 | 空间维度金牌数最多 |
| 🍭 规律小天才 | 规律维度金牌数最多 |
| ✨ 创意小精灵 | 创意维度金牌数最多 |
| ⭐ 星星收集者 | 累计星星数 ≥ 200 |
| 🌈 全能小女王 | 5个维度均达金牌 |
| 💎 传说级天才 | 全部12章所有维度金牌 |

称号显示在主界面头像旁，根据成就动态更新，可在荣誉殿堂查看全部已解锁称号。

## 5. 界面设计

### 5.1 视觉风格

- **整体风格**：梦幻粉彩，温柔渐变，圆角卡片，充满魔法气息
- **配色**：每个世界独立主色，粉彩系（粉红/薰衣草/薄荷/蜜糖/天蓝/彩虹）
- **字体**：圆润可爱字体，大字号，清晰易读
- **动效**：答对时花瓣/星星从天而降，勋章解锁时闪光动画，界面过渡柔滑滑入

### 5.2 向导角色

每个世界配备一位小精灵/仙子角色：
- 题目开始时出现，给出鼓励
- 答对时有特别庆祝反应
- 答错时温柔提示，不批评
- 关卡完成时陪同展示星级

### 5.3 平板横屏布局

**主界面（世界地图）**
- 顶部导航栏：角色信息（精灵头像 + 当前称号）+ 总星星数/勋章数/奖杯数
- 主体：3 × 2 世界卡片网格，大尺寸，视觉冲击强
- 底部导航：首页 / 荣誉殿堂 / 题包中心 / 设置（4栏，大图标）

**答题界面（横屏分栏）**
- 左栏（55%）：题目内容 + 精灵对话气泡 + 题目图示
- 右栏（45%）：4个大选项卡片 + 确认按钮
- 顶部：世界名称 + 题目进度 + 计时器
- 进度条：跟随答题实时推进

**触控适配**
- 选项最小高度 56dp
- 选项间距充足，防误触
- 触控热区适配平板大屏手指操作

### 5.4 页面导航结构

```
主界面（世界地图）
├── 世界详情（上/下学期选择）
│   └── 章节地图（5个维度区域）
│       └── 关卡列表
│           └── 答题界面
│               └── 结算界面（星级 + 动画）
├── 荣誉殿堂（勋章 / 奖杯 / 称号展示）
├── 题包中心（查看 / 加载扩展题包，免费，无购买）
└── 设置（昵称设置 / 音效开关 / 年级设置 / 关于）
```

## 6. 技术架构

### 6.1 技术栈

| 层级 | 选型 |
|------|------|
| 语言 | Kotlin |
| UI 框架 | Jetpack Compose |
| 架构模式 | Clean Architecture + MVVM |
| 本地数据库 | Room |
| 状态管理 | StateFlow + ViewModel |
| 动画 | Lottie + Compose Animation API |
| 导航 | Navigation Compose |
| 最低 API | Android 8.0（API 26） |
| 屏幕方向 | 横屏锁定 |

### 6.2 层级结构

```
app/
├── ui/                    # Composable 界面 + ViewModel
│   ├── worldmap/          # 世界地图界面
│   ├── chapter/           # 章节/维度地图界面
│   ├── question/          # 答题界面
│   ├── result/            # 结算界面
│   └── honor/             # 荣誉殿堂
├── domain/                # 业务逻辑（UseCase）
│   ├── AnswerUseCase
│   ├── ProgressUseCase
│   └── AchievementUseCase
├── data/                  # 数据层
│   ├── local/             # Room 数据库
│   ├── repository/        # Repository 实现
│   └── model/             # 数据模型
└── assets/
    └── packs/             # 内置 JSON 题库文件
        ├── grade1_upper.json
        ├── grade1_lower.json
        └── ...
```

### 6.3 数据库表（Room）

| 表名 | 核心字段 |
|------|---------|
| `level_progress` | pack_id, dimension, level_id, best_stars, attempts |
| `medals` | pack_id, dimension, medal_level, earned_at |
| `trophies` | pack_id, trophy_level, earned_at |
| `profile` | nickname, current_title, total_stars, created_at |

## 7. 范围与边界

**本期包含（MVP）**
- 完整6年级 × 2学期世界结构
- 四选一选择题（主要题型）
- 完整四层荣誉系统
- 平板横屏布局
- 内置一年级完整题库（上下两章，5维度各5关）
- 其余年级题库占位（可后续填充）

**本期不包含**
- 账号系统 / 云同步
- 多用户切换
- 联网功能
- 填空题 / 连线题 / 排序题（二期追加）
- 家长后台报表
- 国际化

---

## 8. 成功标准

- 孩子能在不看说明的情况下自主完成一次完整闯关（可用性）
- 答对时的庆祝动画让孩子有重复答题的欲望（参与度）
- 家长认为界面设计安全无广告（信任度）
- 题目难度符合对应年级认知水平（准确性）
